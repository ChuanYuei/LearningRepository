# 数据类型
使用**type()**查看数据类型
```python
print(type("川月"))
```
查看的是 变量存储的***数据***的类型，因为**变量无类型**，但是他存储的数据有。

# 数据类型转换
int(x) 将x转换为一个整数  
float(x) 将x转换为一个浮点数  
str(x) 将对象x转换为一个字符串
